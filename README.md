# GraphGenerator
Excel to png graph generator for specific risk analysis template

## Link 
Deployed site can be found out https://graphgenerator-app.streamlit.app/
## Template 
Example template available in repository - Sheets should be added that handle calculations for risk catergories and scores. Example template uses random number for illustration purposes.

### Privacy Note: 
This application processes data entirely in-memory. Uploaded files are not stored, logged, or persisted after the browser session is closed.
